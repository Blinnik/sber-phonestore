# Phonestore
Приложение, позволяющее проводить CRUD-операции над сущностью Phone

## Технологии
* Java 11
* Maven
* Lombok (Slf4j для логирования)
* Spring Boot & Spring Web
* JPA (Hibernate)
* JUnit & Mockito
* JavaDoc

## Комментарий
JavaDoc был сгенерирован и находится в src/main/resources/docs.zip
